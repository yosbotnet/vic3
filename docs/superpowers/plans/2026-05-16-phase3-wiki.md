## Per-page template

Every topic page follows the same structure (decision-rule layer on top of mechanics):

```markdown
---
sources:
  - ../../notes/tutorials/NN-slug.md
  - ../../notes/tutorials/MM-slug.md
wiki:
  - https://vic3.paradoxwikis.com/<PageName>
generated_at: 2026-05-16
---

# <Topic name>

## What it is
1–2 paragraphs: definition of the system, when it matters in a campaign, why a
player should care. Strategy-oriented framing — not "here is what this button does".

## Mechanics
Numbered list of the system's core mechanics. Each item: definition (1 sentence)
+ decision rule. Same shape as the source notes' "Core mechanics" section, but
synthesized across multiple notes when applicable.

## Game numbers & rules of thumb
Bulleted list. Generalisable values only (thresholds, modifiers, formulas, ratios).
Pulled from the sources; cite source-note path inline like `[from 02-deficit-spending]`.

## Strategy & playbook
Prose section: 2–5 paragraphs of how to actually play this system in a campaign.
This is the section that adds the most value over the official wiki.

## Common pitfalls
Bulleted accident-traps.

## See also
- Other wiki pages by relative path: `[Trade](../economy/trade.md)`
- Official wiki pages: `[Treasury](https://vic3.paradoxwikis.com/Treasury)`

## Sources
- `notes/tutorials/NN-slug.md` — [video title](url)
- `notes/tutorials/MM-slug.md` — [video title](url)
```

### Task 7: Link-check pass

A common failure mode of parallel agents is inventing wiki page names ("see [Pops](pops/pops.md)" → that file doesn't exist). Programmatically verify every `[label](path)` markdown link inside the wiki resolves.

- [ ] **Step 1: Run link check**

```bash
cd /Users/ybc/code/vic3
python3 <<'PY'
import re, pathlib

WIKI = pathlib.Path("wiki")
LINK_RE = re.compile(r"\[([^\]]+)\]\(([^)]+)\)")
broken = []
for md in WIKI.rglob("*.md"):
    text = md.read_text()
    for label, target in LINK_RE.findall(text):
        # Skip external URLs and anchors-only
        if target.startswith(("http://", "https://", "#", "mailto:")):
            continue
        # Resolve relative to the file
        target_path = (md.parent / target.split("#", 1)[0]).resolve()
        if not target_path.exists():
            broken.append(f"{md.relative_to(WIKI)} -> {target}")
print(f"Checked {sum(1 for _ in WIKI.rglob('*.md'))} files")
print(f"Broken internal links: {len(broken)}")
for b in broken:
    print(f"  {b}")
PY
```

For each broken link: read the offending file, identify the wrong path, use the Edit tool to correct it. Re-run the check.
